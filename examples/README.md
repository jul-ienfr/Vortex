# Manifest Examples

This directory contains example manifests for VORTEX.

| Manifest | Description | Use Case |
|----------|-------------|----------|
| `resell.yaml` | Leboncoin reselling optimization | E-commerce |
| `api-perf.yaml` | API response time optimization | Backend |
| `self-optimizer.yaml` | VORTEX optimizes itself | Meta-optimization |
| `python-project.yaml` | Python project quality | Code quality |
| `docker-project.yaml` | Docker image optimization | DevOps |
| `leboncoin.yaml` | Detailed Leboncoin example | E-commerce |
| `documentation.yaml` | Documentation improvement | Docs |
| `security-audit.yaml` | Security audit | Security |
| `code-review.yaml` | Automated code review | Code quality |
| `database-optimization.yaml` | Database query optimization | Backend |
| `test-coverage.yaml` | Test coverage improvement | Testing |
| `api-documentation.yaml` | API documentation | Docs |
| `performance-profiling.yaml` | Performance profiling | Performance |
| `config-optimization.yaml` | Configuration optimization | Config |
| `dead-code-cleanup.yaml` | Dead code removal | Code quality |
| `dependency-update.yaml` | Dependency updates | Maintenance |
| `error-handling.yaml` | Error handling improvement | Code quality |
| `type-hints.yaml` | Type hints addition | Code quality |

## Quick Start

```bash
# Copy an example
cp examples/resell.yaml my-project.yaml

# Edit the manifest
vim my-project.yaml

# Run optimization
vortex run my-project.yaml
```
